# Структура проекта

## Описание модулей

### `bot/main.py`
Главный файл запуска бота. Инициализирует все компоненты, настраивает обработчики и запускает периодические проверки.

### `bot/config.py`
Конфигурация бота: токены, пути к файлам, настройки API и параметры проверки.

### `bot/handlers/`
Обработчики команд и сообщений:
- `commands.py` - команды `/start`, `/keyboard`
- `messages.py` - обработка текстовых сообщений и кнопок
- `keyboards.py` - клавиатуры для пользователей

### `bot/database/`
Работа с базой данных SQLite:
- `db.py` - функции для работы с группами, ключевыми словами, чатами и статистикой

### `bot/services/`
Сервисы бота:
- `vk_service.py` - работа с VK API (сессии, запросы)
- `vk_checker.py` - логика проверки комментариев
- `excel_service.py` - работа с Excel файлами
- `notification_service.py` - отправка уведомлений в Telegram

### `bot/utils/`
Вспомогательные утилиты:
- `helpers.py` - функции для извлечения ID групп и поиска ключевых слов
- `status.py` - формирование статуса бота

## Файлы конфигурации

- `.env` - переменные окружения (не коммитится)
- `.env.example` - пример конфигурации
- `requirements.txt` - зависимости Python
- `Dockerfile` - образ Docker
- `docker-compose.yml` - конфигурация Docker Compose

## Директории данных

- `data/` - база данных SQLite и Excel файлы (не коммитится)
